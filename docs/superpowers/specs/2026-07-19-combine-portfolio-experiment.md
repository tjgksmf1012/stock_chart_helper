# 사전 등록 실험 ③ — 전략 결합 포트폴리오: 결과 (가설 기각)

날짜: 2026-07-19
상태: 실험 완료 → **가설 기각** (합산 MDD가 개별보다 큼 — 실패 기록)

## 사전 등록 (2026-07-18)

가설: "통과 전략들의 트레이드를 한 계좌(트레이드당 리스크 1% 동일 규칙)로
합산하면, 상관 낮은 전략의 낙폭이 겹치지 않아 합산 MDD < 개별 MDD, EV는 유지."

## 방법

`lab/combine.py` (TDD 9): 리포트 트레이드 복원 + 월별 R 시계열 + 피어슨 상관.
합산 자본곡선은 기존 `risk_based_metrics`(1%/트레이드, −3R 상한) 그대로 재사용 —
개별과 같은 자로 측정. `scripts/run_combine_experiment.py`가 공식 리포트(체제 게이트
구성)의 pass 전략 3개를 결합, 결과는 data/lab_experiments/에 저장.

## 결과 (리스크 1%/트레이드)

| | n | MDD | avg R |
|---|---|---|---|
| tsmom | 534 | 40.6% | 0.45 |
| high52 | 410 | **27.7%** | 0.72 |
| legacy | 1,647 | 41.3% | 0.05 |
| **합산** | 2,591 | **46.1%** ❌ | 0.24 |

월별 R 상관: high52~legacy **−0.19**, legacy~tsmom **−0.02**, high52~tsmom +0.43.

## 해석

- **기각 사유**: 같은 1%/트레이드 규칙으로 합치면 트레이드 빈도가 ~5배 —
  동시 리스크 노출 총량 증가가 분산 효과를 압도한다. 게다가 추세 계열 둘
  (tsmom~high52 +0.43)은 약세 전환기에 같이 나빠진다.
- 합산 누적 수익(+33,212%)은 베팅 횟수 5배의 복리 효과일 뿐이며, 동시 포지션
  자본 제약을 무시한 수치라 실현 불가 — 강조하지 않는다.
- **부수 발견 (후속 실험 후보)**: 상관 자체는 낮거나 음수 — 분산의 재료는
  실존한다. 리스크 총예산을 고정한 결합(예: 합산 시 트레이드당 1%/√3 또는
  전략별 상한)은 별도 사전 등록으로만 검증할 것 — 이번 실험에서 파라미터를
  바꿔 재시도하는 것은 사후 선택이라 하지 않았다.

## 결정

**불채택.** 프로덕션 불변 (전략별 개별 운용·표시 유지). 측정 인프라는 보존.
실험 성적: ① 채택 / ② 기각 / ③ 기각 — 남은 사전 등록 실험: ④ 외국인 수급 필터.

# 사전 등록 실험 ③-b — 리스크 예산 고정 결합 (샤프 비교)

날짜: 2026-07-19 (③의 부수 발견 "상관 낮음"에서 파생 — 계산 전 등록)

## 등록

- 지표: **월별 R 시계열의 샤프 비율(평균÷표준편차)** — 리스크 크기에 불변이라
  "1% vs 1/3%" 같은 스케일 논쟁을 원천 차단하는 유일한 정직한 비교축.
- 가설: 합산(전략 월별 R 합) 샤프 > 최고 개별 샤프. 상관이 낮으니(−0.19~+0.43)
  분모(변동성)가 상쇄되어 개선되어야 한다.
- 판정: 개선 폭이 명확하면(≥ +0.1) 결합 운용 정보를 전략 검증 페이지에 표시 후보,
  아니면 기각. 데이터는 ③과 동일 (공식 게이트 리포트의 트레이드).

## 결과 (2026-07-20)

| | 월별 R 샤프 |
|---|---|
| trend_tsmom | 0.449 |
| high52_breakout | 0.304 |
| legacy_patterns | 0.188 |
| **합산** | **0.474** |

**판정: 기각.** 합산이 최고 개별을 이기긴 하나(+0.025) 등록한 문턱(+0.1)에
크게 미달. 분산 효과는 실재하지만 미미하다 — 추세 계열 둘의 상관(+0.43)이
풀을 지배해서, 낮은 상관(legacy와의 −0.19/−0.02)의 이득이 희석된다.

결론: 결합 운용을 제품에 표시하지 않는다. "하나만 운용한다면 high52"라는
기존 안내(리스크당 수익 0.72R·MDD 27.7% 최우수) 유지. 실험 성적: ① 채택 /
② 기각 / ③ 기각 / ③-b 기각.
